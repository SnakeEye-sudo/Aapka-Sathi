About Aapka-Sathi

Aapka-Sathi is the ecosystem layer that keeps every focused Sathi app connected, discoverable, and consistent.

Why it exists

  • So users know Pariksha Sathi, Rozgar Sathi, Samachar Sathi, and future tools belong to the same family.
  • So one family-level identity and standard can exist across all apps.
  • So future apps can be built faster without repeating product decisions each time.

What it standardizes

  • Shared login
  • PWA install support
  • Day and night theme
  • Mandatory legal and help pages
  • Daily reminders
  • Feedback entry point
